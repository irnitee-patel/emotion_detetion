# emotion_detetion
# 😊 Emotion Detection Dashboard

This project is a real-time emotion detection system using AI and computer vision.

## 🚀 Features
- Real-time face detection
- Emotion prediction (happy, sad, angry, etc.)
- Live camera dashboard using Streamlit
- Emotion analytics and CSV export

## 🛠 Tech Stack
- Python
- OpenCV
- Keras / Deep Learning
- Streamlit

## ▶️ How to Run
```bash
pip install -r requirements.txt
streamlit run dashboard.py
